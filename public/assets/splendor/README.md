# Splendor Assets

Place licensed local card art here to have the Splendor UI render images.

Expected paths:

- `public/assets/splendor/cards/t1_01.png`
- `public/assets/splendor/cards/t2_01.png`
- `public/assets/splendor/cards/t3_01.png`
- `public/assets/splendor/nobles/noble_1.png`

The IDs must match `src/lib/games/splendor/data/cards.ts` and `data/nobles.ts`.
Missing images fall back to the rules-readable card layout.
